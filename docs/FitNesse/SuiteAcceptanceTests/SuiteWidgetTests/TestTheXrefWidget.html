The !see widget takes a page name as argument. It displays the fully qualified page name in bold, prefixed by See:

Comment
Markup Text
Displayed as
!see SomeTestPage See: SomeTestPage[?]

Create some page.

script
start Page Builder
line Dummy Text
page SomePage

Create a suite page with a !see widget

script
start Page Builder
line !see SomePage
page SuitePage
