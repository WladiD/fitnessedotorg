The instructions for each page in the suite are displayed.



Create a Suite page

script Page Builder
line !path lib/*.jar
page SuitePage

Create two sub pages

script Page Builder
line |script|class one|
page SuitePage.TestPageOne


script Page Builder
line |script|class two|
page SuitePage.TestPageTwo
