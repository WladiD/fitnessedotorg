To diplay todays date:

Markup Text
Displayed as
Today is !today. Today is 07 Jul, 2024.
XML is !today -xml. XML is 2024-07-07T18:22:51.
Right now it is !today -t. Right now it is 07 Jul, 2024 18:22.
Tomorrow is !today +1. Tomorrow is 08 Jul, 2024.
Last week was !today -7. Last week was 30 Jun, 2024.
The month is !today (MMM). The month is Jul.

As you can see, you can add or subtract days. You can include the time with -t. And you can use your own format for the time. (See SimpleDataFormat for the syntax.)
