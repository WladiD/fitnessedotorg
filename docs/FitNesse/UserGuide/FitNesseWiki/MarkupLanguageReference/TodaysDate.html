To diplay todays date:

Markup Text
Displayed as
Today is !today. Today is 23 Sep., 2022.
XML is !today -xml. XML is 2022-09-23T10:01:26.
Right now it is !today -t. Right now it is 23 Sep., 2022 10:01.
Tomorrow is !today +1. Tomorrow is 24 Sep., 2022.
Last week was !today -7. Last week was 16 Sep., 2022.
The month is !today (MMM). The month is Sep.

As you can see, you can add or subtract days. You can include the time with -t. And you can use your own format for the time. (See SimpleDataFormat for the syntax.)
