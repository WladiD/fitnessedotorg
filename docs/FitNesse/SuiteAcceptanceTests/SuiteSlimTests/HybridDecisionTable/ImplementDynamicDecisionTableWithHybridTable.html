The Dynamic Decision Table is just a special case of the Hybrid Decision table.

With the following rules for getter and setter the behavior of Dynamic Decision Tables is achieved.
variable defined: SLIM_DT_GETTER=!- { "FormatVersion":"1.0", "MethodExtractorRules":[ { "Scope":".+", "TargetName":"get", "Parameters":"$0" } ] } -!
variable defined: SLIM_DT_SETTER=!- { "FormatVersion":"1.0", "MethodExtractorRules":[ { "Scope":".+", "TargetName":"set", "Parameters":"$0" } ] } -!
