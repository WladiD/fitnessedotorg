SetFixture

SetFixture is the same as ArrayFixture (see ArrayFixture) with one difference: the order of the rows is not checked.

Notes

In Java flow mode, sets returned from the flow fixture method are automatically mapped to a SetFixture

Usage

Use SetFixture instead of RowFixture in Java when working with JavaBeans objects, because it works correctly with JavaBeans getters. Use it instead of ArrayFixture when the order of elements is not important.
