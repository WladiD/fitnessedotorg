A Fixture for summarising the tests run on a page.


The FitFramework provides a fixture that will, when the page is run as a test, print a summary of the test results like so:

Comment
Summary
counts 2 right, 1 wrong, 7 ignored, 0 exceptions
run date Sat Mar 07 15:58:46 GMT 2015
run elapsed time 0:00.10
